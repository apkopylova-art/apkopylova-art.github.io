# Зеленый Барнаул: доступность природных территорий

Интерактивная карта, показывающая распределение зеленых зон и водоемов по Барнаулу, а также зоны пешей доступности к природным территориям.

## Ссылка на карту
[Открыть карту](https://your-username.github.io/green-barnaul/)

## Описание проекта
Проект анализирует доступность природных территорий для жителей Барнаула. Карта показывает:
- Расположение парков, лесов и скверов
- Водоёмы города (включая реку Обь)
- Зоны пешей доступности (500м) от зелёных зон
- Плотность зелёных объектов по регулярной сетке
- Тепловую карту концентрации зелени

## Технологии
- Python 3.13
- OSMnx (выгрузка данных из OpenStreetMap)
- GeoPandas (обработка пространственных данных)
- Folium (интерактивная визуализация)
- Shapely (геометрические операции)

## Источники данных
- **OpenStreetMap** — зеленые зоны, водоемы, жилая застройка
  - Теги: `leisure=park`, `landuse=forest`, `natural=water`, `building=residential`
  - Лицензия: [ODbL 1.0](https://www.openstreetmap.org/copyright)

## Методы визуализации
1. **Хороплет (Choropleth)** — плотность зеленых зон по сетке 500×500м
2. **Тепловая карта (HeatMap)** — концентрация зеленых объектов
3. **Кластеры маркеров (MarkerCluster)** — парки и скверы с названиями
4. **GeoJson-полигоны** — зоны доступности и водоемы

## Как запустить
1. Установите зависимости: `pip install osmnx geopandas folium shapely`
2. Откройте `green_barnaul.ipynb` в Jupyter Notebook
3. Запустите все ячейки
4. Карта сохранится как `index.html`
